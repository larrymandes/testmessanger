### 5. Поток данных правильный

```
IMAP IDLE событие
    ↓
EmailService._startIdleLoop()
    ↓
EmailService._messageProcessor() [установлен из ChatService]
    ↓
EmailService.fetchNewMessages() → fetch с сервера
    ↓
MessageService.processNewMessages() → обработка
    ↓
MessageService._processMessage() → расшифровка
    ↓
StorageService.saveMessage() → сохранение в БД
    ↓
MessageService._notifyUI() → уведомление UI
    ↓
ChatListScreen/ChatScreen callbacks → обновление UI
    ↓
StorageService.getMessages() → загрузка из БД
    ↓
setState() → отображение
```

## 🔍 Критические проверки

### ✅ 1. EmailService НЕ вызывает старые callbacks
**Проверено**: В IDLE loop убран вызов `_callbacks[i]()`

**Было**:
```dart
if (_messageProcessor != null) {
  await _messageProcessor!();
}
// ПОТОМ вызывали старые callbacks
for (int i = 0; i < _callbacks.length; i++) {
  _callbacks[i]();
}
```

**Стало**:
```dart
if (_messageProcessor != null) {
  await _messageProcessor!();
  // Всё! Processor сам уведомит UI через MessageService
}
```

### ✅ 2. MessageService уведомляет UI
**Проверено**: `MessageService._notifyUI()` вызывается после обработки

```dart
Future<void> processNewMessages(List<MimeMessage> messages) async {
  // Обработка...
  if (processed > 0) {
    _notifyUI(); // ✅ Уведомляем UI
  }
}
```

### ✅ 3. ChatService правильно связывает всё
**Проверено**: `ChatService.initialize()` устанавливает processor

```dart
_emailService.setMessageProcessor(() async {
  final maxUID = await StorageService.getMaxProcessedUID(email);
  final newMessages = await _emailService.fetchNewMessages(lastSeenUid: maxUID);
  if (newMessages.isNotEmpty) {
    await _messageService.processNewMessages(newMessages); // ✅ Обработка + уведомление
  }
});
```

### ✅ 4. UI экраны только загружают из БД
**Проверено**: Нет обработки сообщений в UI

```dart
// ChatListScreen
_chatService.addUICallback(() {
  if (mounted) {
    _loadContacts(); // ✅ Просто загружаем из БД
  }
});

// ChatScreen
_chatService.addUICallback(() {
  if (mounted) {
    _loadMessages(); // ✅ Просто загружаем из БД
  }
});
```
