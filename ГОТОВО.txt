  ТЕХНИЧЕСКИЕ ДЕТАЛИ
═══════════════════════════════════════════════════════════

Криптография:
  • ECDH P-256 для обмена ключами
  • AES-256-GCM для шифрования
  • Ephemeral keys на каждое сообщение (forward secrecy)
  • AAD для защиты от replay attacks

Email:
  • IMAP IDLE для мгновенных уведомлений
  • SMTP для отправки
  • Фильтр только [chat] писем
  • Только UNSEEN флаг (не сканирует весь ящик)

Хранение:
  • SQLite для сообщений и контактов
  • SharedPreferences для настроек
  • Приватные ключи только локально

UI:
  • dash_chat_2 (Telegram-style)
  • Тёмная тема
  • Статусы сообщений (🕐 → ✓ → ✓✓)
  • Автонавигация после QR
