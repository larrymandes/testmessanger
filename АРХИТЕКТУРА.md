# Архитектура приложения

## Структура сервисов

### ChatService - главный координатор
**Файл**: `lib/services/chat_service.dart`

Единая точка входа для UI. Связывает все сервисы.

**Методы**:
- `initialize()` - инициализация
- `addUICallback()` - регистрация callback для UI
- `removeUICallback()` - удаление callback
- `sendMessage()` - отправка сообщения
- `fetchAndProcessNewMessages()` - принудительный fetch
- `disconnect()` - отключение

---

### MessageService - обработка сообщений
**Файл**: `lib/services/message_service.dart`

Вся логика обработки входящих сообщений.

**Что делает**:
- Расшифровывает сообщения
- Обрабатывает invite, read receipts, текст
- Сохраняет в БД
- Уведомляет UI через callbacks

---

### EmailService - транспорт
**Файл**: `lib/services/email_service.dart`

IMAP/SMTP подключение и транспорт.

**Что делает**:
- IMAP подключение и IDLE
- Fetch новых писем
- SMTP отправка
- Вызывает processor для обработки

---

### AccountService - управление ключами
**Файл**: `lib/services/account_service.dart`

Генерация и загрузка ключей.

---

### CryptoService - шифрование
**Файл**: `lib/services/crypto_service.dart`

Шифрование/расшифровка, работа с ключами.

---

### StorageService - база данных
**Файл**: `lib/services/storage_service.dart`

Работа с SQLite БД.

---

## Поток данных

```
IMAP IDLE событие
    ↓
EmailService._messageProcessor
    ↓
EmailService.fetchNewMessages
    ↓
MessageService.processNewMessages
    ↓
MessageService._processMessage (расшифровка)
    ↓
StorageService.saveMessage (сохранение)
    ↓
MessageService._notifyUI (уведомление)
    ↓
UI callbacks (ChatListScreen, ChatScreen)
    ↓
StorageService.getMessages (загрузка)
    ↓
setState (отображение)
```

---

## Как использовать

### Инициализация

```dart
final chatService = ChatService(email: email, password: password);
await chatService.initialize();
```

### Регистрация callback

```dart
chatService.addUICallback(() {
  if (mounted) {
    _loadMessages(); // Загружаем из БД
  }
});
```

### Отправка сообщения

```dart
await chatService.sendMessage(
  toEmail: contactEmail,
  encryptedPayload: encrypted,
);
```

### Принудительный fetch

```dart
await chatService.fetchAndProcessNewMessages();
```

## Правила

1. **Вся бизнес-логика в сервисах**
2. **UI только загружает из БД и отображает**
3. **Один источник истины - БД**
4. **Одна точка обработки - MessageService**
