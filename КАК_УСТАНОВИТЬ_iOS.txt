  УСТАНОВКА НА iPhone
═══════════════════════════════════════════════════════════

iOS приложение собирается БЕЗ подписи Apple Developer.
Есть несколько способов установить на iPhone:

═══════════════════════════════════════════════════════════
  СПОСОБ 1: AltStore (БЕСПЛАТНО, БЕЗ JAILBREAK)
═══════════════════════════════════════════════════════════

AltStore позволяет устанавливать неподписанные .ipa файлы.

1. Скачай AltStore на компьютер: https://altstore.io/
2. Установи AltStore на iPhone через USB
3. Скачай app-unsigned.ipa из GitHub Actions
4. Открой .ipa файл через AltStore
5. Готово!

Минусы:
- Нужно обновлять каждые 7 дней через AltStore
- Нужен компьютер для первой установки

═══════════════════════════════════════════════════════════
  СПОСОБ 2: Sideloadly (БЕСПЛАТНО)
═══════════════════════════════════════════════════════════

1. Скачай Sideloadly: https://sideloadly.io/
2. Подключи iPhone к компьютеру
3. Открой app-unsigned.ipa в Sideloadly
4. Введи свой Apple ID (бесплатный аккаунт)
5. Установи на iPhone

Минусы:
- Нужно обновлять каждые 7 дней
- Нужен компьютер

═══════════════════════════════════════════════════════════
  СПОСОБ 3: Apple Developer ($99/год)
═══════════════════════════════════════════════════════════

Если хочешь нормальную установку без ограничений:

1. Купи Apple Developer: https://developer.apple.com/
2. Установи Xcode на Mac
3. Открой проект: flutter_app/ios/Runner.xcworkspace
4. Выбери свой Team в Signing & Capabilities
5. Собери: flutter build ios --release
6. Установи через Xcode на iPhone

Плюсы:
- Работает год без обновлений
- Можно выложить в App Store

Минусы:
- $99/год
- Нужен Mac

═══════════════════════════════════════════════════════════
  СПОСОБ 4: TestFlight (БЕСПЛАТНО, но нужен Developer)
═══════════════════════════════════════════════════════════

1. Нужен Apple Developer аккаунт ($99/год)
2. Загрузи .ipa в App Store Connect
3. Создай TestFlight бета-тест
4. Пригласи себя по email
5. Установи через TestFlight app

Плюсы:
- Работает 90 дней
- Можно пригласить до 10,000 тестеров
