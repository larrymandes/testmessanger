✅ Кросс-платформа (Windows, Android, iOS)
✅ Работает БЕЗ серверов - только IMAP/SMTP

═══════════════════════════════════════════════════════════
  БЕЗОПАСНОСТЬ
═══════════════════════════════════════════════════════════

🔒 Приватные ключи хранятся только локально (SQLite)
🔒 Сервер email видит только зашифрованные данные
🔒 Forward secrecy через ephemeral ключи на каждое сообщение
🔒 MITM защита через QR-коды (out-of-band key exchange)
🔒 AAD защищает от replay attacks и tampering

═══════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════

❌ GitHub Actions не запускается?
   → Проверь что репозиторий публичный
   → Проверь что Actions включены в настройках

❌ Ошибка при локальной сборке?
   → Запусти: flutter doctor
   → Для Windows нужен Visual Studio 2022 с C++ workload

❌ Ошибка IMAP/SMTP?
   → Проверь пароли приложений в Mail.ru
   → Проверь что IMAP/SMTP включены в настройках почты
